
# CG Bot
CapeGroup's Bot

## Installation
Run 

```bash
npm install
```
, then copy example.config.json to config.json and edit everything
to your own values.

## Usage
```bash
node src/index.js
```
and running `#waffle` in any channel to check if it's running.
